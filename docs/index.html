注意事項 (Caution)

  • 計算結果の正確性や再現性を保証するものではありません。 計算結果を使用する際はご自身で検算をお願いします。
  • 浮動小数点で計算を行っている関係上、 「誤差なし」と表示されても実際にはごくわずかな誤差が含まれている場合があります。
  • 素子の数や値の範囲を増やすと探索空間が指数関数的に増加するため、 条件によっては現実的な時間で探索が完了しないことがあります。
  • 問題の報告や要望は GitHub リポジトリ または X (Twitter) へお願いします。
