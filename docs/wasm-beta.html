注意事項 (Caution)

  • 計算結果の正確性や再現性を保証するものではありません。 計算結果を使用する際はご自身で検算をお願いします。
  • 計算をブラウザ上で行っているため、使用する抵抗の範囲を広げたり素子の数を増やしすぎると 探索に時間がかかり、プチフリが発生することがあります。 完全にフリーズするのを避けるため、探索空間が広すぎる場合はエラーにしています。 エラーになった場合は制約条件を狭めてみてください。
  • 問題の報告や要望は GitHub リポジトリ または X (Twitter) へお願いします。
